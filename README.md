# Gamejam

用监控窥探黑暗，躲怪找钥匙逃出生天。

## 游戏简介

这是一款 2D 恐怖潜行探索游戏。玩家被困在怪物游荡的黑暗校园中，需要利用随身的监控平板观察各个房间，搜寻藏在旅行袋中的钥匙与线索，并在危急时躲进床铺、衣柜等掩体。集齐钥匙后，找到校门逃离这场噩梦。

## 玩法目标

- 在黑暗校园中探索大厅、宿舍、教室、厕所等区域。
- 翻找旅行袋，收集逃脱所需的钥匙。
- 使用监控平板观察房间情况，判断怪物和目标位置。
- 避开怪物追击，必要时寻找可躲藏物体暂避。
- 集齐钥匙后前往校门，完成逃脱。

## 操作方式

| 按键 | 功能 |
| --- | --- |
| W / A / S / D | 移动 |
| 鼠标 | 控制角色朝向 |
| E | 互动 |
| R | 打开 / 关闭监控平板 |
| Z / X | 切换监控画面 |
| Alt | 显示 / 隐藏鼠标光标 |
| Esc | 暂停游戏 |

## 游戏特色

- **监控观察机制**：通过监控平板切换房间画面，获取怪物与场景信息。
- **潜行与躲藏**：怪物会巡逻和追击玩家，玩家可以利用床、衣柜等物体隐藏。
- **随机钥匙分布**：钥匙会分布在旅行袋中，提升每次探索的不确定性。
- **黑暗校园氛围**：配合手电、环境音、怪物音效和叙事文本营造压迫感。
- **完整结局流程**：包含开场文本、失败结局、逃脱成功结局和主菜单流程。

## 项目结构

```text
Assets/
  Audios/        音效与背景音乐资源
  FontAsset/     字体资源
  Prefabs/       预制体
  Resources/     运行时加载资源
  Scenes/        游戏场景
  Scripts/       游戏逻辑脚本
  Sprites/       角色、场景和 UI 图片资源
  Tests/         编辑器测试
Packages/        Unity / 团结包配置
ProjectSettings/ 项目设置
docs/            设计与实现文档
```

## 运行方式

1. 使用团结引擎 / Unity 2022.3.62t7 或兼容版本打开项目根目录。
2. 等待项目资源导入完成。
3. 打开 `Assets/Scenes/MainMenu.scene` 或直接运行构建设置中的首个场景。
4. 点击 Play 开始游戏。
